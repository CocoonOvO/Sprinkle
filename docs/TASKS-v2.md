## 开发疏漏回顾

### Phase 2 遗漏的数据库模型

**问题**：`src/sprinkle/models/` 只有 `__init__.py`，没有实际模型

**需要的模型**：
- User（用户）
- Conversation（会话）
- Message（消息）
- ConversationMember（会话成员）

### API 层使用内存字典

**问题**：API 使用内存字典而不是真实数据库

**需要的改动**：
- API 层连接真实 PostgreSQL 数据库
- 实现 CRUD 操作

### 测试质量问题

**问题**：
- 测试只检查状态码
- mock 测试代替真实测试
- 没有验证返回数据准确性

**需要的改动**：
- 重写测试，验证数据准确性
- 使用真实数据库测试
